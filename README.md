## Description

前提としてこのアプリケーションは、余分な機能を省き、より直感的に扱えるアプリケーションを目指した為、  一つのサーバーに対して、一つの団体が利用することを想定して開発されています。  
	
※余分な機能の定義としては、次のようなユーザーが直感的に操作する事ができない機能を仮定します。
1. room作成機能   
	
	
予定をシェアし、その予定に対してコメントをしたり、特定のユーザーの予定を確認する事ができます。
